Introduction

Have you ever heard the term Nintendo Hard? It originated back in the days of the NES when video games were super challenging. A combination of very limited continues, difficult enemies and sometimes clunky controls made many 8-bit games incredibly frustrating to play. Some developers left hidden features in their games to help the player win. These hidden features were called Cheat Codes.

Cheat Codes provided a variety of perks, including extra lives or continues, powered ups, or even allowing the player to start at a specific level. Here are some of gaming's most legendary cheat codes.

How to use Cheat Codes

You will need to be familir with standard game controller inputs. Here are the buttons that will be referenced in this document.

Up, Down, Left, Right, Start, Select, L(left-shoulder), R(right-shoulder), A, B, X, Y

Cheat Codes can be entered in a number of ways:

  • Entering a button combination in quick sequence.
  • Holding a set of buttons for a certain period of time.
  • Entering a specific word in field. eg: The name of your game save.

The Konami Code

The Konami Code is the most legendary cheat code of all time. It appeared in many of Konami's games on the NES and made their games much easier to beat. It is well known for aiding players to complete the notoriously difficult Contra games. At the start screen enter the following button sequence:

Up, Up, Down, Down, Left, Right, Left, Right, B, A

In Contra this gives you 30 extra lives. Good luck, you'll still need it.

Super Mario Bros - Continue

Super Mario Bros for the NES was a really hard game. It was so hard that the developers built in several shortcuts, called warp pipes, into the game so players could skip much of the levels. But did you know you could Continue after seeing the game over screen?

After the game over screen this is what you'll need to do:

Return to the main menu, then hold A and press Start

You'll restart your game at the beginning of the world you died.

NBA Jam - Infinite Fire

NBA Jam is regarded by many as the best basketball video game of all time. During a match if you scored enough points in a row you were on fire. At that point you became an almost unstoppable dunking machine, enabling you to rack up even more points

Of course there's a cheat code that give's the player infinite fire. It goes like this:

Press B seven times before starting a game, and then hold Up, Y, ad B until tip-off

Boomshakalaka... He's on fire!

Mortal Kombat - Blood Code

Mortal Kombat 1 was seen as an extremely voilent game when it was release. It was a fighting game full of blood and gore. When it finally came to home consoles the blood was removed due to industry regulation. However the Genesis version could access blood by entering the code:

A, B, A, C, A, B, B
